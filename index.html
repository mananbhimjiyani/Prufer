               Examinations are crucial in educational institutions. An automated system aims to evaluate students' subjective answers and allocate marks automatically. This system benefits universities, schools, and colleges by streamlining the Assessment process, thus saving time and effort for faculty and staff.

This project focuses on automating the exam Assessment process. Answer papers are scanned, and machine learning techniques are used for handwriting recognition, converting student answers into standard text. Summarization techniques are then applied to create a concise summary of the document's main points.

The system requires administrators to input key answers. Summarized answers are then evaluated automatically based on these key answers. Marks are awarded as a percentage, contributing to overall results calculation.


Key Features

Time Saving

Evaluators have to review numerous answer papers.

Effort Saving

Evaluators need to go through entire answers.

Error Reduction

Evaluators must manually calculate marks.

Fair Assessment

Assessment standards vary among individuals.

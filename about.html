Vision

               Examinations are crucial in educational institutions like colleges and universities. An automated exam paper Assessment system aims to automatically assess subjective answers written by students, providing efficiency to faculties and examination Assessment cells. This system evaluates answers based on predefined key answers, streamlining the Assessment process and ensuring accuracy.

This project aims to automate the exam Assessment process by scanning answer papers and using handwriting recognition techniques to convert student answers into standard text format. Summarization techniques are then applied to create a concise summary of the original document.

The system requires administrators to store key answers in advance. It then automatically evaluates summarized answers based on these key answers, awarding marks accordingly for overall results.


Characteristics

Time-saving

Evaluators have to check numerous answer papers.

Energy-saving

Evaluators have to read through entire answers.

Error Reduction

Evaluators have to manually calculate marks.

Fair Assessment

Assessment patterns vary among individuals.
